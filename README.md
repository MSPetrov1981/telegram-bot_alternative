# t_bot_alt
для проверки работоспособности клонируйте репозиторий к себе. <br>
установите зависимости командой pip install requirements.txt. <br>
пропишите телеграм токен <br>
создайте файл в корневой папке проекта и  <br>

и пропишите следующее :<br>
BOT_TOKEN1=ваш телеграм токен укажите <br>
SMTP_HOST=настройки исходящей почты br>
SMTP_PORT= порт исходящей почты <br>
SMTP_USER= логин исходящей почты <br>
SMTP_PASSWORD=пароль исходящей почты (или временный пароль для приложений, указывается в настройках почты) <br>
EMAIL_TO=s1digital@ya.ru (куда отправлять анкету с заявкой на запись на прием) <br>
и запустите бота командой python bot.py <br>
